# 🕹️ 2D Tank Battle Game

An action-packed, arcade-style 2D Tank Battle game built using Python and Pygame. Command your tank, navigate the battlefield, destroy enemy forces, and survive the onslaught!

---

## 🚀 Features

* **Smooth Controls:** Responsive tank movement and turret rotation.
* **Dynamic Enemies:** AI-controlled enemy tanks that track, hunt, and fire at the player.
* **Destructible Environments & Obstacles:** Tactical maps featuring walls and barriers for cover.
* **Explosion & Visual Effects:** Satisfying particle effects and animations for combat impact.
* **Score & Health Tracking:** Real-time Heads-Up Display (HUD) showing your current health and high score.

---
